# LinguaFlow - 沉浸式网页翻译助手

> 轻量级 Edge 浏览器扩展，提供沉浸式网页翻译体验，支持 LLM / DeepL / 有道三种引擎。

## 功能

### 核心特性

* **沉浸式内联翻译**: 翻译结果直接标注在原文中，不破坏页面布局
* **多引擎支持**: LLM(OpenAI兼容) / DeepL / 有道翻译，可自由切换
* **智能标注**: 自动识别超出词汇等级的单词，标注翻译，适合英语学习
* **词汇等级**: CET-4 / CET-6 / 雅思 / 高考 / GRE，按需选择
* **全文翻译**: 段落级完整翻译，默认走 DeepL，一秒出结果
* **流式输出**: 每个段落独立请求，谁先到谁先显示
* **滚动渐进**: 每 500 词一块，边滚边翻
* **内存缓存**: 同页面二次访问即时显示
* **自动翻译**: 页面加载自动开始，后台标签不翻

## 安装

1. 下载本仓库到本地
2. 打开 Edge，访问 edge://extensions/
3. 开启开发人员模式
4. 加载解压缩的扩展，选择项目文件夹

## API 配置

| 引擎  | 配置  | 用途  |
| --- | --- | --- |
| LLM | API地址 + Key + 模型 | 智能标注 / 单词列表 |
| DeepL | API Key | 全文翻译(默认) |
| 有道  | AppKey + AppSecret | 全文翻译(备用) |

## 使用

1. 点击扩展图标打开设置
2. 选择模式/等级/引擎
3. 填写 API 信息，点测试连接
4. 打开开关，保存
5. 打开网页自动翻译
6. 右侧绿色按钮切换显示

## 架构

` content.js → sendMessage → background.js → fetch(API) → response → DOM `

* API 调用在 background 中进行，避免 CSP 拦截
* 每段落独立请求，结果流式显示
* 初始可视区最多 5 块，滚动每 500 词一块
